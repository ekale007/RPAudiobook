# Referenz-Clips (synthetisch)

Alle mit **edge-tts** erzeugt, als 24 kHz Mono-WAV. Transkript = `ref_text` für OmniVoice-Cloning.

| Datei | edge-tts Stimme | ref_text |
|-------|-----------------|----------|
| `de_narrator_female.wav` | de-DE-KatjaNeural | Es war ein klarer Morgen, als die Geschichte begann. Die Straßen lagen noch ruhig da. |
| `de_narrator_male.wav` | de-DE-ConradNeural | (gleicher Text) |
| `de_protagonist_neutral.wav` | de-DE-AmalaNeural | Ich weiß noch nicht, wohin das führt, aber ich gehe trotzdem einen Schritt weiter. |
| `en_narrator_female.wav` | en-US-JennyNeural | It was a clear morning when the story began. The streets were still quiet. |
| `en_narrator_male.wav` | en-US-GuyNeural | (gleicher Text) |
| `en_gb_narrator.wav` | en-GB-SoniaNeural | (gleicher Text) |

Neu bauen: `npm run tts:omnivoice:refs`
